PWS: Physics in games


2024-2025
As the last project for CoderClass, I've worked with a partner to try to create a somewhat functional physics engine.
The whole idea was to create a simple game, with all the physics code written by us.
It was made with C++, OpenGL, GLFW, and some other supporting libraries.
The project was way out of our skills, so we haven't been able to finish it.
The things that we have implemented are: simple AABB collision detection and resolution,
(the AABB boxes are visible in the video),
3d model loading and animating, and simple shooting physics.

8-bit computer


2024
Following the Ben Eater's tutorial, I tried to recreate his 8-bit computer.
I never got it fully done, but I have made about half of the whole thing.
I have made the clock module, the three registers, the ALU, and partially the bus.
In this state the "computer" can add and subtract simple numbers.
